WELCOME TO CASE MANAGEMENT SYSTEM

CASE MANAGEMENT SYSTEM FOR WOMEN SECURITY




Now a days women security is one of the major issues.Ensuring the dignity and safety of women must be first priority for a society.If any women face problem they may feel uncomfortable and insecure to go to police station. This proposed system provides to file a case in online instead of going to police station. In general after a case has been charged it has to through.so many phases.But, it would be too burdensome for women to go that many times and also it takes a lot of their time. This portal provides login id and password by registering to the website for the women who filed a case against the accused person.From the provided login id they can check their status of case any time by using case id.
